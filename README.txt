Exercise 1. Stack ADT
Run tests_stack_array.py and tests_stack_linked_list.py.
These test the files stack_array.py and stack_linked_list.py respectively.

Exercise 2. Creating world and making the robot move:
Run tests.py. This prints the maze with robot at R, goal at G, walls denoted by |#| and path denoted by |_|.
Reads in world1.dat (included in this package).

Exercise 3. Finding the shortest path:
Run find_path.py. This prints the maze with robot at R, goal at G, walls denoted by |#| and path denoted by |_|.
Reads in world2.dat (included in this package).
